# learn-analog-clock 

🌍 [EN](README.md)

Sebuah aplikasi web edukasional dan interaktif yang dibangun untuk membantu anak-anak dan remaja memahami cara kerja jam analog. Dengan menggeser jarum detik, menit, dan jam secara individual, pengguna dapat melihat secara langsung (real-time) bagaimana satuan waktu saling berhubungan.

## 💡 Latar Belakang & Motivasi

Dengan pergeseran cepat menuju ponsel pintar dan perangkat digital, membaca jam analog dengan cepat menjadi keterampilan yang mulai memudar. Diskusi viral baru-baru ini dan survei informal menyoroti tren yang mengejutkan: **diperkirakan hingga 70% Gen-Z di Indonesia kesulitan membaca jam analog tradisional.** Karena mereka tumbuh di era di mana waktu disajikan secara eksklusif dalam format digital (misalnya, `14:30`), penalaran spasial dan matematika basis-60 yang diperlukan untuk membaca wajah jam analog tidak lagi diperoleh secara alami. Metode pengajaran tradisional sering kali mengandalkan lembar kerja statis yang gagal menjelaskan hubungan mekanis antar jarum jam.

**Proyek ini ada untuk memperbaikinya.** Dengan membagi jam menjadi tiga dial terpisah yang interaktif (Jam, Menit, Detik) dan mengikatnya secara matematis, pengguna dapat secara fisik *merasakan* bagaimana waktu bekerja. Ketika mereka menggeser jarum menit 360 derajat penuh, mereka langsung melihat jarum jam maju secara proporsional, menciptakan momen "aha!" yang intuitif yang menjembatani kesenjangan antara kebiasaan digital dan keterampilan analog.

## 📌 Fitur
- Kuis
- Mode Gelap (Dark mode)
- Mode Latihan dengan menggeser jarum jam
- Multi Bahasa (Indonesia dan Inggris)

## 🚀 Mulai Cepat

Salin dan tempel perintah tunggal ini ke terminal Anda untuk mengkloning repositori, menginstal dependensi, dan memulai server pengembangan sekaligus:

```bash
git clone [https://github.com/Id-Yuu/learn-analog-clock.git](https://github.com/Id-Yuu/learn-analog-clock.git) && cd learn-analog-clock && npm install && npm run dev
```

Setelah skrip selesai berjalan, buka tautan `localhost` yang disediakan di browser Anda untuk mulai menggunakan aplikasi.
